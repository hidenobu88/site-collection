# 🐾 もふもふパーク — 猫と犬のふれあいカフェ サイト

かわいい猫と犬に「エサをあげる・撫でる・おもちゃで遊ぶ」ふれあい体験ができる、架空のふれあいカフェのデモサイトです。

## 見かた

`index.html` をブラウザで開くだけで動きます(ビルド・サーバー不要)。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | ページ本体(ヒーロー / コンセプト / 料金 / 猫・犬スタッフ / ご利用案内) |
| `style.css` | スタイルシート(モバイルファーストのレスポンシブ対応) |
| `script.js` | 猫12種・犬12種のデータ、SVGイラスト生成、ふれあいインタラクション |

## 主な機能

- 平日フリータイム — ¥3,800
- 休日フリータイム — ¥4,400
- 延長 10分ごと +¥250 / おやつカップ ¥400 / 小学生半額・学割10%OFF

### スタッフ紹介(猫12種・犬12種)

猫: マンチカン / スコティッシュフォールド / ペルシャ / アメリカンショートヘア / ロシアンブルー / ノルウェージャンフォレストキャット / ベンガル / ラグドール / シャム / メインクーン / ブリティッシュショートヘア / アビシニアン

犬: ゴールデンレトリバー / ポメラニアン / 柴犬 / トイプードル / チワワ / ミニチュアダックスフンド / ウェルシュコーギー / シベリアンハスキー / フレンチブルドッグ / ボーダーコリー / マルチーズ / ビーグル

各カードのイラストは、種類ごとの特徴(耳の形・毛色・縞やヒョウ柄などの模様・目の色)を描き分けたSVGを JavaScript で生成しています。画像ファイル不要・オフラインでも表示できます。

### ふれあいインタラクション

各スタッフのカードで「🍖 ごはん」「🤚 なでる」「🧶 あそぶ」ボタンを押すと、

- セリフの吹き出しと絵文字エフェクトでリアクション
- 「なかよし度」がアップ(100で「だいすき♥」バッジ)
- なかよし度は `localStorage` に保存され、次回訪問時も引き継がれます
