○×クイズ・パズル・タイマーであそべる、パステルかわいいWebアプリ

Java 21 Spring Boot Thymeleaf PostgreSQL / H2 Flyway
🎮 ブラウザであそんでみる
ディズニー○×クイズ(全20問・毎回ランダム出題)を中心に、ドラッグ&ドロップのパズル、目標タイムぴったりを狙うストップウォッチの3つの遊びを収録。20問すべて答えると、また最初からランダムな順で挑戦できます。本体は管理画面から問題の登録・編集・削除もできる、フルスタック構成のアプリです。
💡 上の「あそんでみる」ボタンからは、ブラウザだけで動くデモ版で実際にプレイできます。本体アプリはサーバー(Java / Spring Boot)+PostgreSQLで動作し、問題データはデータベースで管理されています。

1トップページ

うさぎのマスコット「ミミィ」がお出迎え。あそびたいゲームをカードから選びます。

2○×クイズ

大きな○×ボタンで答える2択クイズ。進捗バーが進み、最後に正解数を表示します。問題はデータベースから出題されます。

3パズル

ピースをドラッグして絵を完成させるパズル。全3ステージ構成です。

4タイマーゲーム

目標のびょう数ぴったりでストップを狙うゲーム。3〜15秒から目標を選べます。

5問題の管理画面

クイズ問題のCRUD(登録・一覧・編集・削除)を備えた管理画面。バリデーションと解説文にも対応しています。

つくりのポイント

  • Java 21 + Spring Boot + Thymeleaf + PostgreSQL のフルスタック構成
  • Flywayによるマイグレーション管理(テーブル作成・初期データ投入を自動化)
  • プレビュー用のH2インメモリDBプロファイルを用意し、DBなしでも起動確認が可能
  • デザインは差し替え可能なトークン方式(tokens.css)で管理
  • スマホ〜PCまでのレスポンシブ対応
