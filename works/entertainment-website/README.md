# TORALUCE PRODUCTION — 芸能事務所ホームページ（サンプル）

画像とテキストを差し替えるだけで完成する、芸能事務所向けの静的サイト一式です。
`index.html` をブラウザで開くだけで動作します（サーバー不要）。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（ピックアップタレントスライダー＋出演情報スライダー） |
| `company.html` | 会社情報（代表挨拶・会社概要・沿革） |
| `talent.html` | 所属タレント（男女タブ切替・各20名） |
| `profile.html` | タレントプロフィール（`profile.html?id=m01` のようにIDで表示） |
| `sns.html` / `sns-detail.html` | SNS投稿一覧／投稿詳細 |
| `news.html` / `news-detail.html` | 最新出演情報（月別タブ）／記事詳細 |
| `audition.html` | 新人オーディション（募集要項・選考の流れ・応募フォーム） |
| `faq.html` | よくある質問（出演依頼・ファンレター宛先・なりすまし注意など） |
| `contact.html` | お問い合わせフォーム |
| `privacy.html` / `disclaimer.html` / `terms.html` | プライバシーポリシー／免責事項／サイトご利用規約 |
| `css/style.css` | 全ページ共通スタイル（レスポンシブ対応） |
| `js/data.js` | **タレント・ニュース・SNSの全データ（基本的にここだけ編集すればOK）** |
| `js/script.js` | スライダー・タブ・ページ描画などの共通スクリプト |

## 差し替え方法

**コンテンツの編集はすべて `js/data.js` の1ファイルで完結します。**

### 1. トップのピックアップスライダー（8名・1人4秒）
`js/data.js` の先頭にある1行を書き換えるだけです。

```js
const PICKUP_TALENTS = ['m01', 'm02', 'm03', 'm04', 'f01', 'f02', 'f03', 'f04'];
```

タレントIDを並べ替える／別のIDに差し替えるだけでメンバー変更完了。
各スライドはクリックするとそのタレントのプロフィールページへ飛びます。

### 2. 所属タレント（40名）とプロフィールページ
`js/data.js` の `TALENTS` 配列を編集します。男性は `m01`〜`m20`、女性は `f01`〜`f20`。
一覧・トップの抜粋・プロフィールページ（`profile.html?id=◯◯`）すべてに自動反映されます。
`hometown`（出身地）・`hobby`（趣味）・`skill`（特技）を追加すると、プロフィールページに個別表示されます。

```js
{ id: 'm01', gender: 'male', name: '蒼井 隼人', kana: 'AOI HAYATO',
  info: '俳優 / 1995.04.12', hometown: '大阪府', hobby: '釣り', skill: '殺陣' },
```

### 3. 最新情報（月別）と記事詳細ページ
`js/data.js` の `NEWS_ITEMS` 配列を編集します。`month: 7` のように掲載月（1〜12）を指定。
一覧（今月のタブが自動選択）・トップの最新3件・詳細ページ（`news-detail.html?id=◯◯`）に自動反映されます。
`talentId` を指定すると、詳細ページとそのタレントのプロフィールが相互リンクされます。

### 4. SNS投稿と投稿詳細ページ
`js/data.js` の `SNS_POSTS` 配列を編集します。投稿はプロフィールページにも自動表示されます。

### 5. 画像
灰色の「NO IMAGE」プレースホルダーは `<div class="noimg"></div>` です。
実画像に差し替える場合は `<img src="images/xxx.jpg" alt="説明">` に置き換えてください。

### 6. SNSの自動反映（実運用時）
現在はサンプルカード表示です。実運用時は `sns.html` 内のコメントに記載のとおり、
X（publish.twitter.com）や Instagram の公式埋め込みコード、
または SNSアグリゲーター（EmbedSocial 等）のコードに差し替えると、
タレントの投稿が自動で反映されます。

### 7. お問い合わせフォーム
現在はデモ動作（送信完了メッセージ表示のみ）です。
実運用時は `contact.html` のフォームに `action` を設定するか、
formrun・Googleフォーム・自社API等の送信先に接続してください。

### 8. 会社名・住所など
「TORALUCE PRODUCTION／株式会社トラルーチェプロダクション」「サンプル住所」等を
一括置換で実際の情報に差し替えてください（各ページのヘッダー・フッターに共通で入っています）。

### 9. フッターのSNSアイコン
各ページのフッターにX・Instagram・YouTubeのアイコンがあります。
`<a href="#"` の `#` を公式アカウントのURLに差し替えてください。

### 10. オーディション・FAQ
`audition.html` の募集要項テーブルと `faq.html` の質問（`<details class="faq-item">`）は
HTMLを直接編集してください。FAQは `<details>` ブロックをコピーすると項目を増やせます。

## レスポンシブ対応
- 〜480px：スマートフォン（タレント2列、ボタン全幅）
- 〜768px：ハンバーガーメニュー表示
- 〜1024px：タブレット（タレント3列）
- 1025px〜：PC（タレント4列）
