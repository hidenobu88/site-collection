# 黒井戸ノ家 ―KUROIDO― 公式サイト(架空)

ジャパニーズホラー(「リング」「らせん」系)の空気感をコンセプトにした、架空のお化け屋敷の公式サイトです。
`index.html` 1ファイルのみで完結しており(CSS/JS/SVGすべて内蔵)、ブラウザで開くだけで動作します。

## コンセプト

- 「一度入ったら、二度と出られない。」をキーメッセージとして、ヒーロー・中盤の帯・フッターで反復
- VHSノイズ/走査線/砂嵐/井戸/黒髪/白い着物の女——という「リング」的モチーフで統一
- 血のドリップ(SVG)をセクション区切りに使用
- 明朝体(游明朝/ヒラギノ明朝)ベースの和風ホラータイポグラフィ

## 演出(JavaScript)

| 演出 | 内容 |
|---|---|
| 警告ゲート | サイト侵入前に「入る/引き返す」を選ばせる警告画面 |
| 画面の明滅 | 不定期に画面全体がフリッカーする |
| 女のフラッシュ | 約25〜60秒に一度、黒髪の女のシルエットが0.1秒だけ映る |
| 生還者カウンター | 数字が時折「彼女」「███」などに乱れる |
| タブ離脱 | 別タブに移るとタイトルが「……どこへ行くのですか。」に変わる |
| 環境音 | 入場ボタン押下後、48Hzのうなりを伴う低音ドローンが小音量で流れる |

## コンテンツ構成

1. 呪いの起源(架空の背景ストーリー)

4. 入場料金

7. 営業案内・交通

## レスポンシブ対応

- 960px以下:カード類を2列に、2カラム構成を1カラムに
- 600px以下:全カード1列、ナビ縦積み、装飾漢字(呪・怨)非表示
- `prefers-reduced-motion` 設定時はアニメーションを抑制
