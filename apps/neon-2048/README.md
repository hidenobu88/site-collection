# NEON 2048 — 無料スライドパズルゲーム

ネオンデザインの 2048 風パズルゲームです。ビルド不要・完全無料・オフラインでも動作します。

## 遊び方

1. `index.html` をブラウザで開くだけで遊べます(サーバー不要)
2. タイルをスライドさせて、同じ数字同士を合体させます
3. **2048** のタイルを作れたらクリア!(その後も続けてプレイ可能)
4. 動かせるマスがなくなるとゲームオーバー

## 操作方法

| デバイス | 操作 |
|---|---|
| スマホ / タブレット / iPad | 画面を上下左右にスワイプ |
| PC | 矢印キー または W / A / S / D |

## 機能

- スコア表示・ベストスコアの自動保存(`localStorage` を使用、端末内にのみ保存)
- 新規タイル出現・合体時のアニメーション
- 2048 達成時のクリア演出、ゲームオーバー判定
- 「新しいゲーム」ボタンでいつでもリセット

## レスポンシブ対応

- スマホ(縦・横向きの両方に最適化)
- タブレット / iPad(768〜1180px で盤面を大きめに表示)
- PC(最大幅を制限して快適な表示)
- 盤面サイズは `dvh` / `clamp()` によりどの画面でも収まるよう自動調整

## 技術構成

- HTML / CSS / JavaScript のみ(単一ファイル、外部ライブラリ・通信なし)
- CSS `transform` によるタイル移動アニメーション(GPU アクセラレーション)
- タッチイベントによるスワイプ判定(しきい値 24px)
